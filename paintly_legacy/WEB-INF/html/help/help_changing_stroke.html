Changing stroke
LiThA-Paint allows you to change stroke type and color used for each patricular object in your document. That helps you to create even more attractive and custom drawings and colorful presentations online. To change stroke type for an object:

1. Select object by clicking it with left mouse button



2. To change stroke color click Set Color button on Tools toolbar (). "Set color" window will appear and allow you to pick color, enter color value in HTML format and set opacity. After setting color you can close window by pressing "Apply" button.



3. To change sroke type click Stroke type button (). You can change stroke width by + and - buttons or by entering value with keyboard. Dashed area section allows to to set stroke pattern with clicking on green/white dots to set style and adjusting amount of dots with + and - buttons.You can set up corner style you want for object (round, bevel or miter), set up miter factor and adjust line caps. After you've done, click "Apply" button to see your changes immediately in drawing area.
