Layers
As any decent vector graphics editor, LiTha-Paint implements features regarding layer manipulation. Each object within your document is considered by LiTha as separate layer, therefore you can change layout of your drawing by changing layers order. Example:

1. Layer order on this drawing is blue, red, green (bottom-to-top)



2. By selecting blue square and pressing (Bring to front) button (all buttons mentioned in this topic are located on Upper toolbar) we'll place it on top-most position.



3. Now red circle is on the bottom. But by pressing (Bring forward) button we can move it one level higher and it will be inbetween blue square and green quadrangle.



4. If we want to send blue square backward, there's a (Send backward) button we can use having square selected. It will move shape one level lower.



5. Now if we'll press (Send to back) button having red circle selected, it will be again on very bottom of our drawing.
