Exporting Image to the Desktop
LiThA-Paint allows you to download your drawing to local computer and save it in JPG, PNG, SVG or TIFF formats. To save document on local computer please follow these simple steps:

1. Click Save button from Upper toolbar



2. Type filename you like to export your drawing with in "Export document" section, select file format from listbox. Please keep in mind that only alphanumeric characters and "_", "-", "!", "." symbols are allowed in filenames.



3. Click "Export" button and download ready-to-use file.
