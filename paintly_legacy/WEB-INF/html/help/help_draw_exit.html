Exit Polygon, polyline and Bezier drawing mode by pressing ESC
When a polyline is being drawn, its last segment is shown in green. To exit the polyline drawing mode, press ESC.
After ESC is pressed, the last segment disappears, the polyline freezes and can be moved, resized, skewed or rotated.
Use the ESC button to exit other modes, for example, of drawing polygons and Bezier lines.
