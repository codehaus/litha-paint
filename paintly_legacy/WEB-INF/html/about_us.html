Litha-Paint (Beta) provides public javascript API that would allow your site to use the same drawing canvas like Litha-Paint do.

Our wiki describes a several simple steps javascript developer should take to use all features that Litha-Paint has.

Example of usage drawing canvas by 3-rd party html page you can find on this demo.html.
