Publishing Image in the Web
LiThA-Paint provides you efficient tools for sharing your drawings or collaboration. It allows you to make any document within your remote home directory visible to other users by assigning URL to those files which are marked by you as shared.

1. Click Save button on Upper toolbar



2. Choose "Publish to WEB" tab



3. Type filename you want to publish your drawing with and select file format from listbox. Please keep in mind that only alphanumeric characters and "_", "-", "!", "." symbols are allowed in filenames.



4. Click "Publish" button. Your file will appear in list below. When you select file by clicking it you'll see URL address in bottom frame. That address is unique for your image and you can provide it to people you want to share that file with. Also, "UnPublish" or "Delete" buttons allows you to stop sharing the selected file or permanently remove it from server.
